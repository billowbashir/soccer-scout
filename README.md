# Soccer Scout

> A app that offers users statistics,team leagues,player stats and fixtures of their favourite sport.

## Author 

> By **Ruben Benji**
> By **Bashir Hassan**
> By **Lewis Mutuma**
> By **Chris Bartoo**

## Description

> This is a web application that offers users a site that of leagues,team and player stats and in game stats plus fixtures and scores

## Technologies Used

> * Python3.6
> * Flask framework
> * Bootstrap and Css
> * PostgreSQL
> * SQLAlchemy

## Setup/Installation Requirements

### Prerequisites

> * Internet access
> * ```git clone https://github.com/bashir-hassan/soccer-scout.git```
> * ```cd Soccer-scout``

#### To install a virtual environment

> * ```python3.6 -m venv virtual``` 
> * ```source virtual/bin/activate```

#### To install all dependencies

> * ```python3.6 -m pip install -r requirements.txt```

## Dependancy Installments

> * pip install python3.6
> * pip install flask
> * pip install flask-bootstrap
> * pip install flask-script
> * pip install flask-wtf
> * pip install flask-migrate
> * pip install flask-login
> * pip install Flask-Mail
> * pip install flask-uploads
